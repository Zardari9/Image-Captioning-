# YOLO & BLIP Computer Vision Project

## Overview
This project demonstrates the implementation of computer vision techniques using:
- YOLO (You Only Look Once)
- BLIT (optimization / rendering techniques)

The notebook contains detection, processing, and visualization workflows.

---

## Features
- Object detection using YOLO
- Efficient image processing
- Visualization of results
- Notebook-based experimentation

---

## Technologies Used
- Python
- OpenCV
- YOLO
- NumPy
- Jupyter Notebook
